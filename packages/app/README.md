# FuelSync App

React TypeScript frontend for FuelSync vehicle expense tracking.

## Setup

```bash
cd packages/app
npm install
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- React Router
- Tailwind CSS
- HeadlessUI
- Heroicons
- Vite

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
